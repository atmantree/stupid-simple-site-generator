Stupid Simple Site Generator

This is a simple static site generator using Pandoc and a tiny Bash script. Inspired in this blog post it build your pages using the header and footer on the templates/ folder.

Usage

  1. Install Pandoc in your computer
  2. Clone this repo
  3. Add your markdown files with .md extension
  4. Run the build_html.sh script
  5. Get your generated files from the _page folder.

Testing your generated pages

  1. in your terminal visit the _page folder.
  2. run this command python3 -m http.server (or python -m SimpleHTTPServer if you are using Python 2)
